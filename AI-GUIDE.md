# AI Assistant Guide - Project Auction

## Project Overview
This is a startup project built by a 6-person team.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js (coming soon)
- Deployment: Vercel (coming soon)

## Folder Structure
- Auction-Project/ → React frontend code
- .github/workflows/ → CI/CD pipeline

## Git Rules (IMPORTANT)
- Never push directly to main
- Always push to Dev branch
- Open Pull Request for every feature
- Wait for 1 approval before merging

## Branch Structure
- main → production ready code only
- Dev → all development work goes here

## How to Use This Project
1. Clone the repo
2. cd Auction-Project
3. npm install
4. npm run dev

## For AI Assistants
- Always suggest code inside Auction-Project/
- Follow React + Vite conventions
- Never suggest direct push to main
- All new features go in src/ folder
